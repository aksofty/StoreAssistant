Параметры

Атрибут Обязательный По умолчанию Описание
src Да — URL скрипта. Домен определяет, куда виджет отправляет запросы.
data-name Нет Ассистент Имя ассистента в заголовке чата.
data-greeting Нет Здравствуйте! Чем могу помочь? Первое сообщение при отсутствии истории.
data-picture Нет SVG-иконка URL изображения аватара ассистента.
data-color-primary Нет #9999CC Основной цвет (кнопка, заголовок, пузырь пользователя).
data-color-secondary Нет #AEAED5 Дополнительный цвет (акценты, hover-состояния).
